记录一下Nmap的常用指令，隔段时间没用都快忘了

Nmap

Nmap 是一个网络连接端扫描软件，用来扫描网上电脑开放的网络连接端、确定哪些服务运行在哪些连接端、并且推断计算机运行哪个操作系统等等。功能集成强大，测试管理利器。

使用笔记

IPv4地址协议中预留了3个IP地址段，作为私有地址，供组织机构内部使用。

  • A类预留地址：10.0.0.0–10.255.255.255
  • B类预留地址：172.16.0.0–172.31.255.255
  • C类预留地址：192.168.0.0–192.168.255.255

另：127.0.0.1 一定程度上等效localhost，为本地回送地址

指定目标

  • 直接连续指定多个目标

  • 用-指定范围

  • IP/number(1~32)

  • 从字典中导入

  • 随机生成选择IP,指定数目为3

  • 排除指定地址

存活探测

  • PING扫描(-sn或者-sP)

  • 越过存活检测(-Pn、-P0、-PN)

端口扫描

  • 端口状态

    STATE open closed filtered unfiltered open|filtered closed|filtered
    状态 开放 关闭 过滤 未过滤 开放或者被过滤 关闭或者被过滤

  • TCP SYN 扫描(-sS)
    SYN扫描(半开放扫描)作为默认的也是最受欢迎的扫描选项，它执行得很快，不易被注意到，它还可以明确可靠地区分状态。

  • TCP connect扫描(-sT),当SYS扫描不可用时这是个比较好的选择

  • UDP 扫描(-sU)

  • 指定端口扫描

信息收集

  • 操作系统探测(-O)
  • 版本信息收集(-sV)
