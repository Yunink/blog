Python踩坑笔记1
Created 2018-04-17| Updated 2020-04-15|Python
Post View:|Comments:

个人近期使用Python时的一些踩坑笔记，简单记录下

Python数据类型

类型 说明 举例
int 整型 无小数点的数: 13、14、-50
float 浮点型 带小数点的数: 3.14、2.9、-9.2
str 字符串 有序的字符序列: "hello",'你好',"3.14"
bool 布尔型 逻辑运算符: True或False
list 列表 有序序列：[100,"hello",-12.7,"你好"]
dict 字典 无序的键值对: {"blog":"yunink","site":"yunink.top"}
tuple 元组 有序且不可变序列: (100,"hello")
set 集合 无序无重复元素如: {"A","B","c"}

Python运算优先级

类型 相对优先级 说明 举例
+ III 加
- III 减
* II 乘
/ II 除
% 取余
// II 整除
** I 指数 如2 ** 3表示2的3次方

Python字符串

类型 说明 举例
len() 获取长度 len(test_str)
find() 查找位置
count() 计数
(not) in 判断包含
replace() 替换

Python转义字符

  • r"……"，在字符串前面加上r表示原始字符
  • 在需要转义的字符前添加\

Python格式化字符串

  • f"str_a{res}str_b"
  • "str_a{}str_b".format(res)
  • "str_a %res_type str_b" % (res)
  • "str_a"+str(res)+"str_b"
