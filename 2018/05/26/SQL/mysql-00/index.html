关于MySQL数据库服务的windows配置笔记

彻底卸载

  • 停止MySQL服务
  • 删除MySQL文件，默认安装目录在C盘
  • 删除注册表中的相关资料

解压安装

  • 在MySQL文件目录中新建my.ini配置文件，内容示例
  • 以管理员权限打开命令终端，进入MySQL下bin文件目录
  • 执行指令安装MySQL

参考记录

同时安装MySQL8

  • 在MySQL文件目录中新建my.ini配置文件，指定不同的未占用端口，内容示例
  • 以管理员权限打开命令终端，进入MySQL下bin文件目录
  • 执行指令安装MySQL，注意初始化指令中需要添加--console

参考记录

终端快捷方式

将MySQL文件bin目录下的mysql.exe分别重命名为mysql5.exe和mysql8.exe,将bin目录添加到path环境变量中，即可在任意目录下使用指令快捷登录。
