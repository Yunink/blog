新建项目

  • 不要使用idea自动下载的jar包，遇到问题不可控，从官网下载导入。
  • 初始自动生成的web.xml过滤器报红，需去掉ng，即可。
  • 配置struts.xml时，添加extends="struts-default"报红，须在facets中勾选strut2-core-2.5.20.jar
