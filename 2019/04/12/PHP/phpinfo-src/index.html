phpinfo信息
Created 2019-04-12| Updated 2020-04-15|PHP
Post View:|Comments:

Tips: 记录下 phpinfo 页面中可以利用的信息

0x01 基本信息

  • PHP
Type Example
Version 5.3.29
extension_dir D:\phpStudy\php53\ext
allow_url_include 远程文件包含(一般 Off)
short_open_tag On(允许形式，且<?=等价于<? echo)
Registered PHP Streams php, file, glob, data, http, ftp, zip, compress.zlib, compress.bzip2, phar

利用phar 反序列化或 phar/zip 协议绕过有后缀的文件包含：

  • System
Type Example
System Windows NT 5G8X3 6.2 build 9200 (Unknow Windows version Datacenter Edition) i586
  • Server
Type Example
Server API Apache 2.0 Handler
SERVER_ADDR 1xx.3x.1xx.2x（真实 IP）
SERVER_PORT 80
REMOTE_ADDR 2xx.1xx.1xx.2xx
SERVER_ADMIN admin@phpStudyxxx
  • Path
Type Example
Configuration File (php.ini) Path C:\Windows
Loaded Configuration File D:\phpStudy\php53\php.ini
