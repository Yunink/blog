如何优雅的提权
Created 2019-07-15| Updated 2020-04-15|SRC
Post View:|Comments:

关于提权的一些姿势套路的小总结，后面有机会再慢慢补充

一、简单理解提权

1、什么是提权

通过各种已知的（大佬们别笑）漏洞和方法手段，尝试提升自己在系统中的操作权限，以便更好的实现控制。

2、提权的目标

windows系统中system权限，linux系统中的root权限，macos的root权限和去掉rootless权限限制

3、提权的途径

  • 系统漏洞提权（Linux、Windows）
  • 数据库提权(MySQL、Mssql)
  • 系统配置错误提权
  • 权限继承类提权
  • 第三方软件/服务提权
  • 获取高权限账号提权
  • WebSevrver漏洞提权

系统漏洞提权
系统漏洞提权一般就是利用系统自身缺陷，使用shellcode来提升权限。为了使用方便，windows和linux系统均有提权用的可执行文件。

  • Windows的提权exp一般格式为MS08067.exe；
    • MS: Micosoft，固定格式
    • 08：2008年发布的漏洞
    • 067：当年度发布的第67号漏洞
  • Linux的提权exp一般格式为2.6.18-194或2.6.18.c

    • 6:
    • 18:
    • 194:

二、Windows提取姿势

  • 查看系统信息
命令 效果 主要用途
systeminfo 打印系统信息 获取系统的补丁信息
whoami /all 当前账户信息 查看用户的名称等
ipconfig /all 当前网络连接 查看设备的网络状况
route print 打印出路由表 获取设备的网络拓扑
apr -a 打印arp表 同上
hostname 打印主机名 获取设备的名称
net user 系统账户信息 获取设备的用户信息
net localgroup 用户组信息 获取用户组的信息
net start 列出启动的服务 查看开启的服务内容
driverquery 列出安装的驱动 查看设备的驱动状态
netsh firewall show config 显示防火墙配置 看心情

五、MySQL提取姿势

1、获取MySQL的账户信息

这一步问题不大，都拿到shell了，这个就顺手的事了，但这里还是做个记录。在cms应用中，一般都包含定义数据库连接的文件，比如common.php

也可以连接webshell通过命令查询

允许MySQL远程连接，除了使用root用户直接授权外，也可以修改MySQL的配置使可以远程连接，即将user表中的host项默认的localhost改为%

2、经典姿势UDF

利用条件：

  • 1、windows2003、windowsXP、windows7
  • 2、拥有mysql的insert和delete权限
  • 3、已知数据库账号密码

  • 利用原理
    UDF,user defined funcion,用户自定义函数。用户可以通过自己增加函数对mysql功能进行扩充，文件后缀为.dll，然后在mysql中使用用户自定义函数以高权限账号执行命令，添加账号进行提权。

  • 查看信息，导出udf

  • udf提权

3、经典姿势MOF

利用条件

  • Windows<=2003
  • mysql在c:windows/system32/mof目录有写权限
  • 已知数据库账号密码

  • 利用原理
    MOF文件是mysql数据库的扩展文件（c:/windows/system32/wbem/mof/nullevt.mof），其作用是每隔五秒就会去监控进程创建和死亡，而且是system权限。我们可以通过mysql将文件写入一个MOF文件替换掉原有的MOF文件，然后系统每隔五秒就会执行一次我们上传的MOF。MOF当中有一段是vbs脚本，我们可以通过控制这段vbs脚本的内容让系统执行命令，进行提权。

六、Mssql提权姿势

mssql是指微软的SQLServer数据库服务器，它是一个数据库平台，提供数据库的从服务器到终端的完整的解决方案，其中数据库服务器部分，是一个数据库管理系统，用于建立、使用和维护数据库。

1、相关信息查询

主要是获取端口服务的开放情况和相关环境的版本信息等。可以参考前面的windows提权姿势的相关部分。

2、找账号密码后提权

一般来说找到了数据库文件后可以解密得到需要的数据库账号密码，破解后寻找远程连接的方法。这个可以参考前面的MySQL提权姿势的相关部分。

七、小总结

一般来说，拿到shell后离权限就一两步的距离了，为了拓展成果，不建议放弃提权，但为了安全考虑，还是不要明目张胆，得有底线。
