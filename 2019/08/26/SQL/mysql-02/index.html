选修了MySQL的课程，重新学习MySQL过程中的一些笔记，温故知新

简单查询

  • 全查(原表)

  • 部分查询

  • 条件查询

  • 查询空值

  • 区间查询

  • 模糊查询

like关键字表示模糊查询，格式为%内容%，%符号表示被模糊内容的位置，如zh%n%g表示模糊查询的内容为zh***n***g等等

  • 分页查询

limit关键字表示分页操作，前面的数表示开始位置，后面的 数表示显示的数据条数，只有一个数字时表示当前分页显示的数据条数

  • 查询过滤

  • 左连接查询

    以左表为准匹配右表中符合条件的数据，没有则以NULL填充

  • 右连接查询

    以右表为准匹配右表中符合条件的数据，没有则以NULL填充

    两表相互匹配符合条件的数据并输出
